House Price Prediction using Linear Regression
=================================================

This project is part of my Machine Learning Internship at Kodbud.
Task-1: Build a regression model to predict house prices based on features like area, bedrooms, and location.

-------------------------------------------------

Project Overview:
- Implemented Linear Regression to predict house prices.
- Performed data preprocessing (handling missing values, encoding categorical variables, scaling).
- Visualized relationships between features and target variable.
- Evaluated model performance using metrics like R² Score and Mean Squared Error (MSE).

-------------------------------------------------

Technologies & Libraries:
- Python
- NumPy
- Pandas
- Matplotlib / Seaborn
- Scikit-learn

-------------------------------------------------

Project Structure:
House-Price-Prediction/
│── Task-1_House_Price_Prediction.ipynb    # Google Colab with code
│── requirements.txt                       # Required Python libraries
│── README.txt                             # Project documentation
│── dataset/                               # Dataset used from sklearn.datasets

-------------------------------------------------

How to Run:
1. Clone this repository:
   git clone https://github.com/LohithKambhampati1807/House-Price-Prediction.git
   cd House-Price-Prediction

2. Install required libraries:
   pip install -r requirements.txt

3. Open the Google Colab and run all cells:
   Task-1_House_Price_Prediction.ipynb

-------------------------------------------------

Results:
- Predicted house prices with Linear Regression.
- Gained understanding of regression models and how preprocessing improves performance.
- Visualization helped in better feature insights.
